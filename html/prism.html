Точки перелому

Точки перелому (breakpoints) — це специфічні ширини області перегляду (viewport), при яких дизайн веб-сторінки змінюється для забезпечення оптимального відображення на різних пристроях. Ось три найпоширеніші точки перелому та їхні значення:

1. Перша точка перелому (мобільні пристрої):

  • Зазвичай використовується для смартфонів.
  • Типовий діапазон: від 320px до 600px.
  • Приклад медіа-запиту:

CSS Code


      @media (max-width: 600px) {
  /* стилі для мобільних пристроїв */
}

2. Друга точка перелому (планшети):

  • Використовується для пристроїв середнього розміру, таких як планшети.
  • Типовий діапазон: від 600px до 900px
  • Приклад медіа-запиту:

CSS Code


      @media (min-width: 600px) and (max-width: 900px) {
  /* стилі для планшетів */
}

3. Третя точка перелому (настільні комп'ютери):

  • Використовується для великих екранів, таких як монітори настільних комп'ютерів.
  • Типовий діапазон: від 900px і більше.
  • Приклад медіа-запиту:

CSS Code


      @media (min-width: 900px) {
  /* стилі для настільних комп'ютерів */
}
